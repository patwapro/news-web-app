# News Web App
**ReactJs** based news web app with newsapi.
## Prerequisites
The things you need to install.
```
NodeJs and Git(optional)
```
## Installation
Just clone or download the news-app
### development
run 
```
npm install
```
then 
```
npm start
```
### production
run 
```
npm install
```
then 
```
npm run build
```
after build complete
```
serve -s build
```
